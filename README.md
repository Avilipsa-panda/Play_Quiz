# Quiz Player System

This project is a Quiz Player System developed using ASP.NET Web Forms and C#.  
It allows users to enter a Quiz ID to load and attempt quizzes dynamically.

## Features
- Enter Quiz ID to load quiz
- Display quiz information
- Timer-based quiz system
- Multiple-choice questions
- Automatic result calculation
- Score storage in database

## Technologies Used
- ASP.NET Web Forms
- C#
- HTML
- CSS
- JavaScript
- SQL Server

## Project Workflow
1. User enters Quiz ID
2. System fetches quiz details from database
3. Questions are displayed
4. Timer starts for the quiz
5. User submits answers
6. System calculates score and stores result
